Understanding Feature Metrics

  • Power: Average absolute contribution (higher = stronger influence on decisions)
  • Effect: Directional bias - BULLISH (mostly positive), BEARISH (mostly negative), or NEUTRAL
  • Contribution: Average contribution value (positive = bullish, negative = bearish)
  • Usage: Number of times feature was calculated and used in decisions
  • Distribution: Count of positive/neutral/negative signals from this feature
